# エラー内容
- CapsLockの設定が逆になる
## 原因
- １．ショートカットキー(SupperKey+Space)をCapsLockオフの状態で押す
- ２．CapsLockオンの状態でカーソルで入力文字を変更する
- ３．たまに↑ のショートカットキーを多様してても起こる→　Virtulboxのエラー

## 解決策
- １．同じくショートカットキー(SupperKey+Space)をCapsLockオフの状態で押す
- →　その先で通常に戻ってるならカーソルで変更する
- ２．CapsLockのオンの状態で変更した先（日本語入力が多い）でCapsLockをオフにしてカーソルで入力文字を変更する
- Virtualboxのエラー時は↓
- CapsLockオンやオフの状態で別windowに移動する
- CapsLockを何回か押す
- ログアウト
